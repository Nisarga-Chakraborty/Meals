# Meals
Discover delicious recipes, explore cuisines, and personalize your meals — all in one sleek Flutter app.
## Why You’ll Love It
- Personalized experience: Set your dietary preferences (Gluten‑Free, Lactose‑Free, Vegan, etc.) and instantly filter meals.
- Favorites at your fingertips: Save the dishes you love and revisit them anytime.
- Explore global cuisines: Italian, Spanish, Asian, and more — a world of flavors awaits.
- Beautiful visuals: High‑quality food images make browsing fun and appetizing.
- Detailed insights: Tap on any meal to learn more about ingredients, preparation, and cultural background.
  ## Built With Passion
This app was crafted entirely in Flutter, showcasing cross‑platform navigation, clean architecture, and reusable widgets. It’s lightweight, fast, and designed to give users a smooth experience whether on Android or iOS.
## What’s Next
- Cloud sync & backend integration for persistent favorites
- User accounts with authentication
- Admin panel for adding new recipes dynamically
## Try It Out
Clone the repo, run it, and start exploring meals tailored to your taste. Whether you’re a foodie, a developer curious about Flutter, or someone looking for inspiration in recipe apps — this project is for you.
